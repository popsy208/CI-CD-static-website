# 🚀 CI/CD Static Website Deployment with GitHub Pages

A modern DevOps project demonstrating a complete **CI/CD pipeline** for deploying a simple static web application built with **HTML, CSS, and JavaScript** using **GitHub Actions** and **GitHub Pages**.

This project showcases:
- Automated CI/CD workflow
- Continuous deployment with GitHub Actions
- Static website hosting on GitHub Pages
- Clean project structure
- Beginner-friendly DevOps practices
